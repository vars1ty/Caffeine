<h2 align="center">
    What is Caffeine?
</h2>

<p align="center">
    Caffeine is an anti-detect minimalist browser, built using `wry` and `webkit` which makes it mimic Safari on a Mac.
    <br>
    It's extremely bare-bones for now and is mainly made as a hobby project.
</p>
